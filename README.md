# React + TypeScript + Vite
Tune Shelf : This project is a small React application built with TypeScript and Chakra UI, designed to showcase artist albums and songs using the Last.fm API.

## Setup and Run Instructions
## Prerequisites
You need a recent version of Node.js and npm installed on your system.

## Install dependencies
- Configure .env file using .env.sample
- Run `npm i` in client directory

### Api Document
```bash
https://www.last.fm/api#getting-started
```

### Running the Application
```bash
npm run dev
```

### Run unit test
```bash
npm test
```
